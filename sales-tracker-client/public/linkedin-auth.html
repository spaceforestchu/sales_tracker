This page will extract your LinkedIn cookies and upload them to enable job scraping.

Step 1: Make sure you're logged into LinkedIn

Open LinkedIn.com in another tab and verify you're logged in.

Step 2: Extract ALL LinkedIn cookies and session info

Run this command in LinkedIn's Developer Console:

  1. Open LinkedIn.com in a new tab (make sure you're logged in)
  2. Open Developer Tools Console:
    • Windows/Linux: Press F12 or Ctrl + Shift + I, then click Console tab
    • Mac: Press Cmd + Option + I, then click Console tab
  3. Copy and paste this entire command into the console and press Enter:
copy(JSON.stringify({
  cookies: document.cookie.split('; ').map(c => {
    const [name, value] = c.split('=');
    return { name, value, domain: '.linkedin.com', path: '/', secure: true, httpOnly: name === 'li_at' };
  }),
  userAgent: navigator.userAgent,
  platform: navigator.platform
}, null, 2))
  1. After running the command, you'll see "undefined" in the console - that's normal! The data has been copied to your clipboard.
  2. Paste the copied data into the box below and click Upload.
